# 黎昕彤个人作品集

一个可继续扩展的白底编辑式个人作品集首页，展示交互、产品、三维、视觉、信息、机械、空间与 AIGC 实验项目。

## 技术架构

- Vite：本地开发与静态构建。
- 原生 HTML、CSS、JavaScript：保持首版轻量，项目数据独立维护。
- GitHub Actions：推送到 `main` 后自动构建并发布 GitHub Pages。

## 本地运行

```bash
pnpm install
pnpm dev
```

## 测试与构建

```bash
pnpm test
pnpm build
```

## 部署

推送到 `main` 分支后，GitHub Actions 自动部署。仓库首次使用时，需要在 Settings → Pages 中将 Source 设为 GitHub Actions。

媒体地址统一通过 Vite 的站点基础路径生成，保证部署在 `https://liiizncu.github.io/portfolio/` 子目录时图片与视频正常加载。

## 已完成功能

- 首屏先展示姓名、身份、彩色个人照片和核心资料，更多经历可展开查看。
- 8 个项目使用可选择的编辑式展厅，一次聚焦一个项目。
- 每个项目按“问题—过程—方案—结果”组织完整叙事。
- 从原始 PPT、PDF 中提取用户画像、用户旅程、信息架构、草图、结构图和测试过程辅助讲解。
- 使用用户提供的 VERSA 空气炸锅演示视频与塔罗制作过程视频。
- SportLoop 接入 6 组来自原始 PPT 的真实交互动效，项目主媒体在电脑和手机一屏内完整显示。
- SportLoop 的界面截图与交互动图统一使用 iPhone 16 Pro Max 比例、圆角外框和灵动岛展示。
- 桌面端使用左侧项目索引，手机端使用可展开项目菜单。
- 纯白底、黑灰文字和少量东方暗红、金色、蓝色点缀。
