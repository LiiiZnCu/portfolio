# 黎昕彤个人作品集

一个可继续扩展的白底编辑式个人作品集首页，展示交互、产品、视觉、信息、机械、空间与 AIGC 实验项目。

## 技术架构

- Vite：本地开发与静态构建。
- 原生 HTML、CSS、JavaScript：保持首版轻量，项目数据独立维护。
- GitHub Actions：推送到 `main` 后自动构建并发布 GitHub Pages。

## 本地运行

```bash
pnpm install
pnpm dev
```

## 测试与构建

```bash
pnpm test
pnpm build
```

## 部署

推送到 `main` 分支后，GitHub Actions 自动部署。仓库首次使用时，需要在 Settings → Pages 中将 Source 设为 GitHub Actions。

## 已完成功能

- 首屏个人档案、精选项目、详细介绍与联系方式。
- 完整展示 7 个项目与移动端导航。
- 使用用户提供的塔罗制作过程视频。
- 桌面与手机响应式布局。
